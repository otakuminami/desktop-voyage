# 桌面大航海

一个桌面浏览器上的轻量航海生存网页游戏。玩家通过设定目标自动航行，在随机事件中做决策，目标是尽可能活得更久。

## 当前阶段
- 阶段：MVP 骨架已实现（可运行）
- 平台：桌面浏览器优先
- 技术栈：原生 `HTML/CSS/JavaScript`
- 存档：`localStorage` 全自动存档（单存档位）

## MVP 核心玩法
- 2D 自由海图（东亚/东南亚真实地图）
- 地图有固定边界，船只与路线规划都不会越界
- 选择港口后自动航行
- 15-45 秒随机触发事件，事件弹窗期间航行暂停
- 事件选项影响 `耐久/补给/金币`
- 无 Game Over：船毁或补给耗尽后触发救援，休整 30 分钟后再出航
- 存活时间在救援期间暂停；救援结束后重置并重新计时

## 核心数值
- 开局：`耐久 100 / 补给 100 / 金币 0`
- 补给：随时间自动消耗
- 耐久：仅由事件变化
- 金币：仅由事件变化

## 本地运行
1. 安装依赖：`npm install`
2. 启动静态服务：`npm run serve`
3. 打开浏览器访问：`http://127.0.0.1:4173`

## 当前控制方式
- 通过右侧面板选择港口并点击 `设为港口目标`
- `F`：切换全屏

## 测试快捷键（开发验证用）
- `A`：立即触发一个随机事件
- `B`：立即触发救援休整
- 事件弹窗中按 `Space`：选择第一个选项
